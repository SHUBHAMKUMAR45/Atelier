# Font Assets

Place these font files in this directory before running the mobile app:

## Required Files
- `Cormorant-Regular.ttf` — https://fonts.google.com/specimen/Cormorant
- `DMSans-Regular.ttf`    — https://fonts.google.com/specimen/DM+Sans
- `DMSans-Medium.ttf`     — https://fonts.google.com/specimen/DM+Sans
- `DMMono-Regular.ttf`    — https://fonts.google.com/specimen/DM+Mono

## Quick Download
```bash
npx expo install expo-font
# Then download fonts from Google Fonts and place in this directory
```

The app gracefully falls back to system fonts if these files are absent.
